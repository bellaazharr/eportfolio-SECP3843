# Reflection — Data Analytics Programming (Assignment 2)

## Objective
Apply data analytics programming techniques in a structured notebook-based assignment, completed in a group.

## What I Did
Contributed to a group notebook assignment applying analytics programming concepts from the course.

## How This Connects to Later Coursework
Working in a shared notebook format previewed challenges I'd face again in group data engineering work — keeping a notebook reproducible and understandable for teammates is a different skill than just getting code to run for yourself.

## What I'd Do Differently
I'd push the group to add more markdown explanation between code cells while we worked, rather than after — notebooks that make sense in the moment don't always make sense to someone (including future me) reading them later without that context.
