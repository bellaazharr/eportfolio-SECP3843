## 1. Group Project: Web Crawling — Mudah.my
**Role:** Member 1 (Group Lead) — crawling Vehicles and Property categories, report introduction, system design
**Status:** ✅ Submitted

Built a crawler targeting Mudah.my. After BeautifulSoup and Selenium proved unreliable on the live site, switched to using Mudah.my's internal search API directly.

**Design note:** The pivot from traditional scraping (BeautifulSoup/Selenium) to using the site's internal API wasn't a workaround — it was a more robust solution, since it avoided the fragility of parsing rendered HTML that changes between page loads.
